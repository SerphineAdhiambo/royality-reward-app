# 🎉 Vibe Loyalty Rewards App

A simple, joyful loyalty tracking app for local salons, barbershops, and eateries — built during the **Vibe Coding Hackathon** for the **#1MillionDevs Movement**.

## 💡 Problem Statement

> Local businesses lack systems to retain loyal customers.  
> 
> Many don’t track customer visits, and as a result, they miss out on creating lasting customer relationships.

## ✅ Our Solution

We created a **lightweight loyalty rewards web app** that allows local business owners to:

- Track customer visits using their phone numbers
- Reward frequent customers after a configurable number of visits (default: 5)
- Store data securely using Supabase
- Send automated (mocked) SMS notifications when rewards are unlocked

---

## 🔧 Tech Stack

| Component | Tool |
|----------|------|
| Frontend | HTML + Tailwind CSS + JavaScript |
| Backend  | [Supabase](https://supabase.com) (Auth + DB) |
| Optional API | Twilio or other SMS gateway |
| AI Ideas | Claude.ai for prompt-based customer insight (future feature) |

---

## 📦 Features

- ✅ Add new customers with phone numbers
- 🔄 Log repeat visits
- 🎁 Automatically notify when a reward is earned
- 📊 Simple and responsive UI
- 🔐 Secure data storage with Supabase
